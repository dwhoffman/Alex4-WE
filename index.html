Alex the Allegator 4 - web edition


A WIP HTML5 remake of the classic "Alex the allegator 4" game, available here and created by Johan Peitz from free lunch design. Alex 4 WE, includes the first 6 levels of the original game, and has been developed using the melonJS library, a free HTML5 game engine.

This is a Chrome Experiment.

For those familiar with the original game, please keep in mind that this is a WIP remake, meaning that the game won't be totally identical to the original game and lacks a few things since I did not have access to the original game stuff (e.g. no intro, only 6 levels for now, sprites/animations not 100% accurate)

Fore more informations about how to play and credits for the original game, please see the following page.

Enjoy :)
